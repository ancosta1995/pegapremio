# Fonts Directory

## Attack Of Monster Font

Para usar a fonte "Attack Of Monster" no site, você precisa:

1. Baixar o arquivo da fonte `Attack Of Monster.ttf`
2. Colocar o arquivo nesta pasta (`public/fonts/`)
3. O arquivo deve se chamar exatamente: `Attack Of Monster.ttf`

### Onde baixar:

Você pode baixar a fonte de sites como:
- Google Fonts (se disponível)
- Font Squirrel
- Dafont
- 1001 Fonts

Ou se você já tem o arquivo, apenas copie para esta pasta.

### Verificação:

Após colocar o arquivo, verifique se o caminho está correto:
- Arquivo deve estar em: `public/fonts/Attack Of Monster.ttf`
- Acessível via URL: `http://localhost:8000/fonts/Attack Of Monster.ttf`
